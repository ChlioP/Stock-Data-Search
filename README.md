# Stock Data Search

This project is a Flask-based web application that provides comprehensive information about companies, stocks. The application fetches data from the Tiingo API and presents it in a user-friendly manner.

## Features:

- **Company Outlook:** Provides information about a company's name, ticker symbol, exchange code, start date, and description.
- **Stock Summary:** Shows details such as trading day, opening price, closing price, high and low prices, volume, and changes.

## Setup:

- The user first opens a web page as shown below in Figure 1, the initial search page, where he/she can enter a stock ticker symbol. 
- Providing a value for the “Stock Ticker Symbol” field is mandatory.
- The search form has two buttons: search button and clear button.
- There are 2 tabs: Company Outlook and Stock Summary.

### Prerequisites:

- Python  Python programming language and Flask framework.
- Creating web pages using HTML, CSS, JavaScript, DOM, JSON format and
XMLHttpRequest object. 
- A Tiingo API key
